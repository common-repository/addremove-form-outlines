== Description ==

This plugin will add styles to your header based on how you want to style your form element's outline. You can change the color, style, width, and offset... or competely disable it. Going so far as to duplicate the style across your admin panel too!

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
